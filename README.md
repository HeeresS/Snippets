# Small-Code
Small codes that will help with daily tasks for Linux and Windows. 
### Add text to line
~~~
awk  '{print "http://" $1}' urls.txt
~~~
### Change permissions of folder (Linux)
~~~
sudo chmod +x $(find /PATH/)
~~~
